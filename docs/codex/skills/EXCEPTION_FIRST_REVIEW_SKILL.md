# Exception-First Review Skill

## Purpose

Minimize manual review by focusing reviewer attention on exceptions and
promotion candidates.

## Rules

- Review extraction failures, missing citations, low-confidence labels, and
  ambiguous source language first.
- Surface possible contradictions and items that may require actuarial
  judgment.
- Include proposed learner-facing promotions in the review packet.
- Do not make the reviewer inspect every chunk when a concise exception packet
  is enough.
