# RAG Readiness Skill

## Purpose

Structure outputs so future retrieval-augmented generation can work from
approved indexed material.

## Rules

- Include stable ID, domain, module or topic label, source title, source
  reference or path, and page or section reference when available.
- Preserve chunk text, summary, keywords, confidence, review flags, and
  learner-facing eligibility status.
- Future answers should cite approved indexed material and say when that
  material does not support an answer.
- Keep approved and unapproved material clearly separated.
