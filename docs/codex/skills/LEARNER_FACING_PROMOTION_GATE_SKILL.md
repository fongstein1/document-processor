# Learner-Facing Promotion Gate Skill

## Purpose

Treat all processing outputs as draft candidates until they clear promotion
gates.

## Rules

- Default every chunk, summary, glossary item, quiz item, and lesson point to
  review-only status.
- Promote only when source citation exists, support is clear, wording is not
  misleading, confidence is high, and no unresolved review flags remain.
- Keep anything uncertain in review output only.
- Never let generated wording outrank source-bound evidence.
