# Regulatory Profile

Use this profile when the source is regulatory or quasi-regulatory text.

## Optional fields

- `authority_level`
- `requirement_type`
- `jurisdiction`
- `effective_date`
- `supersession`
- `regulatory_framework`
- `filing_or_reporting_requirement`
- `compliance_implication`

## Guidance

- Keep binding text separate from commentary or implementation notes.
- Use stable source locators for sections, page windows, and citations.
- Flag repeal, withdrawal, supersession, or companion-only status explicitly.
- Prefer a small number of authoritative fields over a sprawling taxonomy.
