# Liability-Modeling Profile

Use this profile for reserve, liability, and projection-model documents.

## Optional fields

- `reserve_framework`
- `model_platform`
- `projection_component`
- `assumption_type`
- `prescribed_vs_company_assumption`
- `model_input`
- `model_output`
- `validation_control`
- `model_limitation`

## Guidance

- Separate prescribed assumptions from company or prudent-estimate assumptions.
- Preserve projection context and model limitations.
- Keep model-platform detail optional and portable.
- Use this profile as an extension of the generic core, not as a replacement.
