# Governance Profile

Use this profile for governance, control, approval, review, and evidence
documents.

## Optional fields

- `process_owner`
- `control`
- `evidence`
- `review_frequency`
- `approval`
- `exception`
- `issue`
- `remediation`

## Guidance

- Keep control evidence separate from narrative explanation.
- Preserve review frequency, approval, and exception handling where present.
- Use this profile for model governance, document governance, and similar
  control-oriented material.
