# Reporting Profile

Use this profile for reporting, filing, and production-control documents.

## Optional fields

- `reporting_period`
- `report_owner`
- `source_system`
- `reconciliation`
- `exhibit`
- `deadline`
- `reviewer`
- `exception`
- `downstream_use`

## Guidance

- Keep filing and reconciliation context separate from narrative explanation.
- Preserve deadlines, owners, and exceptions where present.
- Use this profile for reporting procedures, exhibits, and control documents.
